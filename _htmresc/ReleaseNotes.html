STM32CubeN6 Firmware Package V1.2.0 / 04-June-2025

Main Changes

Second maintenance release of STM32CubeN6 firmware package supporting STM32N6xx devices

Drivers

- CMSIS device drivers supporting STM32N6xx devices

- HAL/LL drivers available for all peripherals

- BSP drivers supporting NUCLEO-N657X0-Q and STM32N6570-DK boards

Middlewares

- MW ST Filex V6.4.0 support

- MW ST Levelx V6.4.0 support

- MW ST Netxduo V6.4.0 support

- MW ST ThreadX V6.4.0 support

- MW ST USBX V6.4.0 support

- MW External Memory Manager V1.4.0 support

- MW External Memory Loader V1.1.0 support

- MW Open BootLoader V1.0.0 support

- MW Image Signal Processing (ISP) V1.0.2 support

- MW H.264 Video Encoder support

Projects

- Templates and Templates_LL projects based on HAL/LL APIs

- HAL and LL examples for IP peripherals

- Video Encoder applications

- USB-PD applications

- Open Bootloader applications

- Root of Trust (ROT) applications

- AzureRTOS 6.4.0 applications
